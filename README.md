# 🎓 College Management System

## 📌 Project Overview
The College Management System is a web-based application built using Flask that helps manage various college operations such as student details, staff management, attendance, marks, and administrative tasks.

This system provides an organized way to handle academic data efficiently.

---

## 🚀 Features

- 👨‍🎓 Student Management (Add, Update, View)
- 👩‍🏫 Staff Management
- 📝 Attendance Tracking (Student & Staff)
- 📊 Marks Management
- 📅 Exam Scheduling
- 💰 Salary Management (for staff)
- 🔐 User Authentication (Login system)
- 📂 Admin Dashboard

---

## 🧠 How It Works

1. User logs into the system (Admin/Staff)
2. Admin can manage students, staff, exams, and records
3. Staff can update attendance and marks
4. Data is stored and retrieved using a database
5. All operations are handled through a Flask backend

---

## 🛠️ Tech Stack

- **Backend:** Python (Flask)
- **Frontend:** HTML, CSS
- **Database:** SQLite
- **Libraries:** Flask, SQLAlchemy 
- **Tools:** Git, GitHub, VS Code

## ▶️ How to Run

### 1. Clone the repository

### 2. Install dependencies

pip install -r requirements.txt


### 3. Run the application

python app.py


### 4. Open in browser

http://127.0.0.1:5000
